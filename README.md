# Install steps:  
```sh
sudo apt-get install vim exuberant-ctags cscope  
git clone https://github.com/bearlin/vim_trinity.git
cp -r vim_trinity/.vim ~/.vim
cp -r vim_trinity/.vimrc ~/.vimrc
```
